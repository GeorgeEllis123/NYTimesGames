## üéØ Implemented Game: Connections

**Connections** is a daily word game where players group 16 words into four sets of four based on a shared theme.

### Gameplay Rules
- 16 words are presented in a grid
- Players must find 4 correct groups of 4 related words
- Only one valid solution exists
- Players have a limited number of incorrect guesses

### Features
- Accurate rule enforcement
- Word selection and grouping logic
- Feedback for correct and incorrect groupings
- Clean, minimal UI inspired by the original game
- Fully playable start to finish
